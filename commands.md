# Commands
1. We have 10 options 
2. # Option 1 - **Login**
    we login into an already existing database using roll number if the roll number doesn't exist it gives an error
3. # Option 2 - **Register**
    we create a new user using his name and roll number
4. # Option 3 - **Search by date**
    we show all the logs occured on a specific date
5. # Option 4 - **Delete a user**
    we remove user's data from the database using the roll number
6. # Option 5 - **View logs**
    recent activities done by the admin
7. # Option 6 - **Display registered students**
    it displays the registered students
8. # Option 7 - **Check status**
    we enter a student's roll number and checks his/her current status
9. # Option 8 - **Time difference field search**
    we input a time duration in seconds to display the data of users who have been exited for longer than the specified time
10. # Option 9 - **Number of students registered**
    it outputs the number of students registered
11. # Option 10 - **Exit**
    we exit the program
